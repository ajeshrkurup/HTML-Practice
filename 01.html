  • Link to another website address eg: href="http://www.google.com"
  • Link to a local page, ie to a page with in the website eg link to a contact page, eg: href="contact.html"
  • Link within the page, ie if our page is so long, we can add link to top of the page or other location in the page eg: href="#top"
